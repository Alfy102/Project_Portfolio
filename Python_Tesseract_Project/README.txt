Current implementation only covers:
1) Basic uploading of file.
2) Basic retrieving of uploaded file.
3) Basic tesseract conversion.
4) Added filename exploit protection.
5) Fixed file name spacing issue by replacing space with underscore.


Future implementation:
1) Add support for new line of line break for teseract conversion.
2) Add support for logging of uploaded files.
3) Add support for basic image editing like rotate or flip.
4) Add support for batch conversion. The text will be compiled in different pages that corresponds to the file number.
5) Add support for PDF Tesseract.
5) Add support for clearing uploaded files in server for privacy safety.
